What important permissions do we request?

Here we explain all the important permissions that our app requests. These permissions are necessary for our app to perform certain features.

· Access information about networks

This permission allows Smart Cleaner to check the user's current network status and perform the corresponding advertising strategy and this permission is required for Data Thieves feature to help users master the use of background data usage.

· Collect battery statistics

This permission is used to get the battery usage data, in order to provide power consumption ranking in Battery Saver feature.

· Access your camera

This permission is used to provide a flashlight toggle in the notification bar and take photos of the intruders who want to snoop on your private apps.

· Clean cache of applications

This permission is required to clean the cache of installed applications on your device in Junk Clean feature. 

· Collect component usage statistics

This permission is required to identify the applications that need to be locked for Virus Cleaner function and help users to manage apps usage 

· Disable the keyboard 

This permission allows us to provide quick entry for features like Boost or Junk Clean on the lock screen page.

· Find out the space used by any package

This permission is used to find out the space used by any package so as to provide Junk Clean feature. 

· View the running apps

This permission is used to view the running apps in order to clean up memory usage.

· Open network sockets

This permission is used to access the device’s network for downloading Virus Database.

· Kill background processes

This permission is used to stop other apps and boost your phone.

· Directly call phone numbers & answer phone calls

This permission allows users to directly answer or hang up the phone calls.

· Phone state

This permission is used to avoid interrupting the user's call status from Cleaner auxiliary function and allows Call Blocker to get call status.

· Broadcast after the system finishes booting

This permission is used to help Phone Clean Apps run automatically when the phone is restarted.

· Create windows shown on top of all other apps

This permission is used to make sure the super battery saver function can be used normally and show the process of it.

This permission is used to make sure the Phone Clean Apps function can be used normally. 

· Use fingerprint hardware

The only purpose which we used this permission is to help users achieve fingerprint unlock function.

· The vibrator

This permission allows Phone Clean Apps to provide vibration feedback after wrong password input.

· Keep processor from sleeping or screen from dimming

This permission is allowed to avoid screen dimming from the process of virus scan and junk scan. 

· Read and write phone storage space

Mounting and unmounting file systems.

These permissions are used to detect and delete junk files on your devices and also used to read and store photographs if you use the Photo Vault feature.

· Accessibility

This permission is used to close apps chosen by users which can speed up users’ devices and also we need this permission to provide power saving mode for users to prolong battery life.

· Get, authenticate and manage accounts on the device Read and write sync settings

These permissions are used to help users retrieve their passwords through your accounts on their devices. We don’t have any access to your password.
